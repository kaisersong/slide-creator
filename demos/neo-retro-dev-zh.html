05 / 08
参考
命令
/slide-creator 交互式：样式发现 → 内容 → 生成。最适合从零开始。
/slide-creator --plan 创建 BRIEF.json IR。在生成 HTML 前停止；如需给人审阅，再派生 PLANNING 视图。
/slide-creator --generate 读取 BRIEF.json，跳过提问，生成 HTML。从 IR 快速到演示文稿。
/slide-creator --export pptx 通过系统 Chrome 截屏每张幻灯片 → 组装 .pptx。无需 Node.js。
