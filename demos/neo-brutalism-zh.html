kai-slide-creator

三个命令，生成 21 种风格的幻灯片
无需模板、无需导出、无需平台

0 依赖 21 预设 纯 HTML
核心痛点

为什么现有的演示工具总是不够好？

  • 01
    工具链太复杂 — 安装依赖、配置环境、维护版本，做一次 PPT 要折腾半天
  • 02
    通用 AI 美学泛滥 — 所有幻灯片长得一样：渐变、圆角、紫色调，缺乏辨识度
  • 03
    平台锁定风险 — 用了某个平台就离不开，文件导不出来，格式私有化
核心理念

看图选风格，而不是描述风格

3
生成 3 个风格预览
→
用眼睛选最满意的

种设计预设可选

工作流程

三个命令，从想法到演示

  • 01
    /slide-creator — 告诉 AI 你想做什么，系统自动生成规划
  • 02
    选风格 — 看图选一个最喜欢的预设，或描述你想要的感觉
  • 03
    打开演示 — 生成纯 HTML 文件，浏览器直接打开，零安装
/slide-creator --plan "为 Acme v2 产品发布制作路演 deck" /slide-creator --generate
核心功能

四个关键能力，覆盖完整工作流

播放
F5 / ▶ 全屏播放

自动缩放适配屏幕，控制栏自动隐藏

编辑
E 键进入编辑模式

contenteditable 实时修改，Ctrl+S 保存

备注
P 键打开演讲者模式

备注、计时器、页数、翻页同步

导出
/kai-html-export 导出 PPTX

像素级 PNG 或原生可编辑模式

安装方式

一行命令，零依赖运行

# Claude Code 告诉 Claude：安装 kai-slide-creator
# OpenClaw clawhub install kai-slide-creator

只需要 Python 3，无 npm / node / 构建步骤
