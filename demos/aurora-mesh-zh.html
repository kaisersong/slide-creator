slide-creator

为 Claude Code 和 OpenClaw 打造零依赖 HTML 演示文稿

21 预设风格
0 外部依赖
∞ 幻灯片
01
Problem

演示工具越做越复杂，结果千篇一律

  • 复杂工具链 — 安装配置半天，还没开始做内容
  • 通用 AI 审美 — 每份演示看起来都一样，缺乏品牌辨识度
  • 平台锁定 — 离开特定软件就无法查看或编辑
  • 模板同质化 — 选来选去都是那几个通用模板
02
Discovery

看图选风格，而非描述风格

Bold Signal
数据驱动、高对比
Aurora Mesh
专业品牌、毛玻璃卡片
Neon Cyber
开发者、赛博朋克
Swiss Modern
研究、思想领导力
03
Solution
21 预设风格 每种都有独立视觉签名
100% 浏览器运行 零服务器，零依赖
3min 快速出稿 从想法到完整演示
04
Workflow

三个命令，从想法到演示

01
创建规划
/slide-creator --plan "产品发布"
02
选择风格
从 3 个预览中直观选择最匹配的视觉风格
03
生成并打开
/slide-creator --generate
05
Features

四个核心能力，覆盖完整工作流

视口适配
每张幻灯片精确填满视口，clamp() 响应式字号，零溢出
浏览器内编辑
E 键进入编辑模式，实时修改文字和备注，Ctrl+S 保存
PPTX 导出
图片模式像素级还原，Native 模式文字可编辑
两阶段规划
自动模式快速出稿，精修模式深度规划叙事结构
06
Install

一行命令，立即开始

Claude Code
安装 https://github.com/kaisersong/slide-creator
OpenClaw
clawhub install kai-slide-creator

Ready?
