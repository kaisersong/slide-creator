Edit Mode

Edit in the browser.
Save to disk

E

Toggle Edit

Hover the hotzone top-left. Click Edit. All text becomes editable.

D

Speaker Notes

Notes panel slides up from bottom. Notes sync to presenter view.

⌘S

Save File

Ctrl+S downloads the edited HTML. Self-contained, zero dependencies.

Getting Started

Three commands.
Zero friction

/slide-creator

Interactive mode — describe your mood, pick a style, get a deck

/slide-creator --plan

Generate outline — review and edit before building

/slide-creator --generate

Build from BRIEF.json — full HTML with all modes
