What is slide-creator?

A skill that generates self-contained HTML presentations. Zero external dependencies. Runs in any browser. 21 distinct visual styles — each one intentional, none generic.

  • No frameworks, no build step
  • 21 design presets, not one generic theme
  • Playback mode — F5 fullscreen
  • Presenter mode — notes, timer, sync
  • Edit mode — E key, Ctrl+S save

Enter edit mode

Hover the top-left hotzone. Click ✏ Edit. All text becomes contenteditable. Live notes panel at the bottom.

CTRL + S

Save to file

Edits persist via browser download. The HTML you edit is the final artifact — no build step, no export.

Edit mode is always enabled by default. Every generated HTML is a living document.

Two planning depths — match the deck to the stakes

Auto

Fast draft

3–6 minutes. Quick outline, immediate generation. For internal reviews and quick iterations.

Polish

Deep planning

8–15 minutes. Structural narrative, visual lock per slide, 16-point review auto-executed.
